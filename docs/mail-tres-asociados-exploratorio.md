# Mail a Tres Asociados — Exploración de datos operacionales

**Para**: [contactos Tres Asociados]
**Asunto**: Consulta — cómo manejan los datos operacionales de Viña y Curicó

---

Estimados,

Espero que se encuentren bien. Me contacto porque estamos iniciando un proyecto de consolidación de toda la información operacional de nuestros fondos (PT, Apoquindo, TRI) en una base de datos centralizada. El objetivo es contar con un repositorio único desde el cual podamos analizar el desempeño histórico de cada activo, automatizar reportes y dashboards, y tomar decisiones con mejor información.

Como ustedes administran Viña Centro y Curicó, quería explorar con ustedes cómo manejan hoy la información operacional (ingresos, gastos, cobranza, etc.) para ver si hay una forma eficiente de vincular esos datos con nuestro sistema.

Algunas preguntas simples para ir evaluando:

1. ¿En qué sistemas o herramientas guardan ustedes los datos operacionales mensuales de Viña y Curicó? (ej. Excel, un ERP, un software de administración de propiedades, etc.)

2. **¿Tienen acceso directo a una base de datos SQL** donde está la información operacional? Si es así, ¿sería posible que nosotros accedamos a esa BD (con las credenciales y permisos correspondientes) para traer los datos directamente?

3. ¿Con qué periodicidad ustedes cierran mes? (ej. ¿el ER de cada mes está disponible el día 5 del mes siguiente?)

4. Si nosotros dijéramos "necesitamos los datos operacionales mensuales de Viña y Curicó de forma automática y continua", ¿cuál sería el camino más práctico desde su infraestructura actual?

No hay urgencia en responder ni compromiso de nuestra parte aún — solo queremos entender cómo operan ustedes para poder evaluar opciones realistas de integración.

Quedo atento a cualquier pregunta.

Saludos,
